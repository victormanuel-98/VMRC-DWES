## Se debe:

### Ejercicio 1
Crear un servidor en Node.js que devuelva **“Hello World!”** cuando se acceda al puerto **4000**.

### Ejercicio 2
Crear un servidor en Node.js que devuelva una **página web** al acceder al puerto **3000**.

### Ejercicio 3
Crear un servidor en Node.js que tenga distintas páginas según la **URL accedida**.

### Ejercicio 4
Añadir dos rutas principales:
- `/page` → Devuelve una página web.  
- `/error` → Devuelve una página de error con el **código 404**.

### Ejercicio 5
Modificar el servidor para que las páginas se carguen desde **ficheros .html**, en lugar de escribir el HTML directamente en el código.

### Ejercicio 6
Enviar el parámetro `name` por **query string** y devolver:  

`http://localhost:3000/?name=Victor` → **Hello Victor!**

### Ejercicio 7
Crear una ruta `/fizzbuzz` que reciba un número por **query string** y devuelva toda la secuencia FizzBuzz hasta dicho número.

`http://localhost:3000/fizzbuzz?num=15`

## Entregables

### En clase
- Ejercicios **1, 2 y 3**

### Tarea
- Ejercicios **4, 5, 6 y 7**
